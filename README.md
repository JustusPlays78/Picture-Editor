# Photo Editor

## General
This Project is build with VueJS, TSC, Tailwindcss. It is generally build in Darkmode. There will be no Lightmode here.

## Installation & Setup
### npm

``npm i``
``npm run dev``

## yarn

``yarn install``
``yarn run dev``
